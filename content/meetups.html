PeruJUG Meetups

Join Our Community

PeruJUG hosts regular monthly meetups where Java developers and enthusiasts gather to share knowledge, learn new technologies, and network with fellow developers.

Monthly Meetups

Our meetups typically feature:

  • • Technical presentations on Java technologies
  • • Hands-on workshops and coding sessions
  • • Networking opportunities with local developers
  • • Discussions about industry trends and best practices
  • • Q&A sessions with Java experts

Past Meetups
